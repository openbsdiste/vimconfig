# VIMCONFIG : configuration de vim

Mode d'emploi :

```
    cd ~
    rm -rf .vim .vimrc
    git clone https://github.com/openbsdiste/vimconfig .vim
    ln -s .vim/vimrc .vimrc
    cd .vim
    git submodule init
    git submodule update
```
